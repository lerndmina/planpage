Widget gallery — everything interactive planpage can do

2026-07-19 planpage living docs

Every interactive widget the planpage chrome provides, in one page. Agents: author the markup shown in each section — the behavior, persistence, and styling are injected automatically. Everything you tick, pick, or type here saves in your browser; the Copy state button above serializes all of it as markdown to paste back to an agent.

Checklist

Markup: <ul class="check"> with plain <li> items (<li data-checked> pre-checks). The counter chip in this heading is <span class="count" data-count="demo-check">.

  • Tick me — then reload the page and I stay ticked
  • Items can hold code, bold, and links
  • This one was pre-checked at publish time
  • Ticking updates the sticky progress bar at the top

Step tracker

Markup: <ol class="steps track"> — the normal step list, but the number badge is a button: click it to cycle pending → doing → done → blocked.

  1. Write the fragment
    Author inner HTML using the widget classes.
  2. Publish it
    The chrome injects the runtime when it sees widget markup.
  3. Work the plan
    Statuses persist here and feed the progress bar.
